# Proyect-TERA
"Project Tera" es un título de supervivencia y acción donde el jugador controla a un autómata en un entorno marciano hostil. La experiencia se centra en defender infraestructuras biológicas contra oleadas incesantes de enemigos, forzando un equilibrio entre el riesgo de moverse para recolectar recursos y la necesidad de proteger posiciones clave.
